
# Splunk SOC Home Lab

## Overview
This project demonstrates a beginner-level Security Operations Center (SOC) home lab using Splunk Enterprise to collect, monitor, and analyze Windows event logs.

The lab uses Splunk to ingest and investigate logs from Windows Event Viewer, helping simulate basic SOC monitoring and detection workflows.

------------------------------------------

## Objective
To understand SIEM fundamentals, log ingestion, event analysis, and threat detection using Splunk.

------------------------------------------

## Architecture

Windows Endpoint → Event Viewer → Splunk → Search Queries → Detection → Investigation

-------------------------------------------

## Detection Use Cases

### Failed Login Detection
- Event ID: 4625
- Generated by entering incorrect Windows password.

### Successful Login Detection
- Event ID: 4624
- Used to validate successful authentication.

### Process Creation Monitoring
- Sysmon Event ID: 1
- Used to analyze parent-child process relationships.

### PowerShell Monitoring
- Tracks command execution.

### Firewall Monitoring
- Tracks allowed/blocked connections.

### Defender Monitoring
- Tracks antivirus activity.
